# Installation

#### NPM

``` bash
npm install @laozei/normalize.css
```

#### YARN

``` bash
yarn add @laozei/normalize.css
```

#### PNPM

``` bash
pnpm install @laozei/normalize.css
```

---

# Example

#### When using with `js`

```js
import "@laozei/normalize.css";
```

#### When using with `css`

```scss
@import "@laozei/normalize.css";
```
